## Design System

### Typography
- **Headlines**: Playfair Display (bold serif)
- **Body**: Inter (clean sans)
- **Data/Labels**: Space Mono (monospace)
- **NO system fonts**

### Colors
- Background: Warm off-white (#faf9f7)
- Text: Near-black (#1a1a1a)
- Accent: Terracotta (#c73e1d)
- Borders: Sharp black

### Components
- **Buttons**: Sharp corners, 3px border, 4px shadow offset
- **Cards**: Bento-style, sharp corners, hover lift
- **Inputs**: Sharp borders, focus accent
- **Badges**: Border + background, uppercase labels

---

## Key Improvements

1. **NO Tailwind defaults** - All custom CSS variables
2. **NO rounded corners** - Sharp edges throughout
3. **NO gradients** - Solid colors only
4. **NO glassmorphism** - Flat, raw aesthetics
5. **Proper hierarchy** - Editorial typography scale
6. **Asymmetric layouts** - Breaking the grid intentionally
7. **Monospace for data** - Technical, raw feel
